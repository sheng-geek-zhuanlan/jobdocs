   <ClassPath location="C:\Jar\myJava.jar"/>

2. Develop Guide: How to Develop Java Job on AC

3. Job API

  • Function
  • Description
  • void reportPass(String msg)
  • report pass status to AC with detail description
  • void reportFail(String msg)
  • report fail status to AC with detail description
  • void setOutputValue(String paraname,String paravalue)
  • output parameter to AC
  • String getInputValue(String paraname)
  • get parameter value by name from AC
  • String getEnvProperty(String paraname)
     
  • get env parameter value by name from AC, the env variables were loaded by <PropFile> or <Property>
  • String getConfProperty(String paraname)
     
  • get config parameter value by name from AC, the config variables were loaded by <ConfFile>
  • String getDataProperty(String paraname)
     
  • get test datavalue by name from AC, the test data were loaded by <TestData>
  • void beginTestCase(String casename)
  • define a new test cases with case name
  • void endTestCase()
  • mark the test case to be end
  • void beginTest(String testname)
  • define a new test with test name
  • void endTest()
  • the name of parameter
  • void outputFrameLog(String msg)
  • Write log into AC framelog
  • String writeIntoACChannel(String paraname,String paravalue)
  • Write data into AC global data channel
