2 在AC上开发运行QTP测试：2-1 Global Variables
Variables
  • Variable
  • Description
  • HOME
  • The framework root directory.
  • dicGlobalData
     
     
  • A dictionary (key-value pair) which holds all stuffs read from data_global.xls, data_config.xls and user provided data files.
  • TIMEOUT
     
  • The time to wait before considering an object does not exist. The default value is 20 seconds.
  • strSuitName
  • The name of the running test suite.
  • strCompName
  • The name of the component the test is running against.
  • strCaseName
  • The name of the current test case
  • strCaseFile
  • The name of the test case vbs file
  • strCurDir
  • The current working directory
  • strLogFileName
     
  • The log file name, which is used in AddToTestLog utility function.
